This is Blue's Dynamic Lights, made to be used as a cosmetic simple dynamic lights mod that uses vanilla shaders to accomplish it's goal.
This mod allows colored dynamic lights with up to 512 dynamic lights in render distance
This is compatible with optifine but not with optifine shaders.

The mod on curseforge can be found here https://www.curseforge.com/minecraft/mc-mods/blues-dynamic-lights
